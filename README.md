 Weather App

Weather App — это веб-приложение для отображения погоды в любом городе мира. Пользователь может быстро узнать текущую погоду и прогноз на 5 дней.

---

 Цель проекта

- Предоставить пользователю удобный способ узнать актуальную погоду.  
- Отображать температуру, влажность, скорость ветра и погодные условия.  
- Показывать прогноз на 5 дней для планирования активности.

---

 Технологии и архитектура

Проект выполнен с использованием **HTML, CSS и JavaScript**:

- HTML— структура приложения: заголовки, поле поиска, блоки с текущей погодой и прогнозом.  
- CSS — стилизация интерфейса: тёмная тема, градиенты, адаптивная верстка, анимации.  
- JavaScript — логика приложения: поиск города, работа с API, динамическое обновление данных.  

---

Работа с JavaScript

В JavaScript реализованы следующие функции:

- *Поиск города* и получение координат через Geocoding API.  
- *Получение текущей погоды* и прогноза на 5 дней через Open-Meteo API.  
- *Отображение информации*: температура, влажность, ветер, состояние погоды.  
- *Обработка ошибок*: город не найден, проблемы с подключением.  
- *Обновление интерфейса* с текущим временем запроса.  
- Использование *DOM API* для работы с элементами и *Event API* для обработки событий (клики, нажатие Enter, ввод текста).

---

 Работа с API

Приложение использует два внешних API:

1. *Geocoding API* — определение координат города по названию.  
2. *Open-Meteo API* — получение текущей погоды и прогноза на 5 дней.  

Данные запрашиваются асинхронно через *fetch()*, обрабатываются и отображаются на странице. В случае ошибки выводится сообщение пользователю.

---

 Почему выбрал этот проект

- Практическое приложение, которое полезно каждому.  
- Демонстрирует работу с API и асинхронный JavaScript.  
- Наглядное приложение: пользователь сразу видит результат поиска.  
- Проект легко расширяется: можно добавить графики температуры, прогноз на больше дней, данные о давлении и т.д.
